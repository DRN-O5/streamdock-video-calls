<h1>Online Real-Time Video Calling Application</h1>
<h2>Overview</h2>
A full-stack real-time video calling application designed for one-to-one and group communication. The application focuses on secure access, low-latency interaction, and a responsive user interface while handling real-time communication efficiently.

<h2>Libraries & Frameworks Used</h2>

React – Builds the component-based frontend user interface.

Node.js – Handles server-side logic and request processing.

Express.js – Manages RESTful APIs and backend routing.

MongoDB – Stores user and application-related data.

GetStream.io – Provides real-time video and communication features.

Tailwind CSS – Utility-first CSS framework for responsive styling.

DaisyUI – Prebuilt UI components built on Tailwind CSS.

Zustand – Lightweight state management for client-side state.

Axios – Handles HTTP requests between frontend and backend.

JSON Web Tokens (JWT) – Secures authentication and user sessions.

CORS – Enables safe cross-origin communication.

Toast Notifications – Displays real-time feedback for user actions.
